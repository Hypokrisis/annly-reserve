                        ✨ CÓMO PERSONALIZAR
═══════════════════════════════════════════════════════════════════════════════════

1️⃣  Cambiar prompt:
   Dashboard → Asistente IA → Prompt personalizado
   (El AI usará EXACTAMENTE el prompt que ingreses)

2️⃣  Cambiar servicios:
   Dashboard → Servicios → Agregar/Editar
   (El AI mostrará los precios y duraciones reales)

3️⃣  Cambiar horarios:
   Dashboard → Horarios → Configura días y horas
   (El AI calculará disponibilidad en tiempo real)

4️⃣  Cambiar link de reserva:
   Dashboard → Asistente IA → Enlace de Reservas
   (Al confirmar cita, enviará TU link personalizado)

5️⃣  Ver conversaciones:
   Supabase SQL Editor → SELECT * FROM bot_message_logs
   (Analiza qué preguntan los clientes, mejora prompt)
